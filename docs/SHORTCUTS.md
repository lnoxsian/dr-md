# Keyboard Shortcuts

The app uses the platform command modifier, so `Ctrl` on Linux and Windows maps to `Command` on macOS.

## File

- `Ctrl+N` - New note
- `Ctrl+Shift+N` - New folder
- `Ctrl+O` - Open folder
- `Ctrl+S` - Save
- `Ctrl+Shift+S` - Save as
- `Ctrl+W` - Close note

## Editing

- `Ctrl+Z` - Undo
- `Ctrl+Shift+Z` or `Ctrl+R` - Redo
- `Ctrl+A` - Select all

## Markdown

- `Ctrl+B` - Insert bold markers
- `Ctrl+I` - Insert italic markers
- `Ctrl+K` - Insert link markers
- `Ctrl+Shift+C` - Insert code block markers
- `Ctrl+L` - Insert checkbox markers
- `Ctrl+/` - Toggle comment markers

## Views And Layout

- `Ctrl+1` - Editor mode
- `Ctrl+2` - Preview mode
- `Ctrl+3` - Split mode
- `Ctrl+E` - Toggle file explorer
- `F11` - Toggle focus mode

## Navigation

- `Ctrl+P` - Quick open / search files in explorer
- `Ctrl+Tab` - Switch to next tab
- `Ctrl+Shift+Tab` - Switch to previous tab
- `Alt+Left` - Switch to previous tab
- `Alt+Right` - Switch to next tab
